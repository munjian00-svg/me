# moonjian
안녕하십니까 00부서 마케팅에 지원하게 된 문지안이라고 합니다. 저는 대학시절 제1전공으로 철학을 전공하였고, 그 중에서도 논리를 분석하고 체계화하는 학문인 논리학에 관심이 있었습니다.
논리학은 쉽게 설명할 수 있는 문장이 있다. "비가 오면 땅이 젖는다. 지금 땅이 젖어 있다. 그러므로 비가 왔다." 라는 문장에서 땅이 젖은 이유는 누군가 물을 뿌렸을 수도 있고, 수도관이 터졌을 수도 있기 때문입니다. 결론이 반드시 참이라고 보장할 수 없다. 이를 논리학에서는 전건 긍정의 오류라고 하는데 이처럼 무언가의 논리를 분석하여 찾는 학문이므로 저는 수많은 오류들을 찾는 과정이 어려운 문제에 답을 찾아내는 것처럼 탐구하는 재미가 있었고 따라서 관심을 가지게 되었습니다. 이를 기반으로 요즘 시대에 떠오르는 키워드인 AI와 연관 지어 연구한다면 어떨까 싶었고 그 주제는 AI는 과연 여러가지 오류중에 무엇의 오류로 해석하기 어려운 문장이 있으면 그걸 사람한테 "00오류가 있습니다"가 아닌 이 문장은 "00오류와 ##오류 중에 하나로 해석하는 게 좋을 것 같으나 딱 무엇의 오류인지 해석하기에는 어렵습니다"로 대답할 수 있을까? 였습니다. 제가 여태까지 봐 온 AI는 사람이 질문하면 00이다로 답을 내려서 대답해주는 형식이었기 때문에 애매한 문제에도 정확하지 않는 대답을 내릴 가능성이 있기 때문에 이를 연구주제로 삼고 싶었습니다. 그래서 5년뒤 
저는 이 연구주제의 해답을 찾아 제가  하고 싶은 마케팅 전략인 논리학이 기반인 마케팅 전략에 계속해서 발전중인 AI를 통합시켜 더 많은 사람들을 끌어당기는 마케팅전략을 펼쳐보고 싶습니다.
